# PlayableLighting

Mod commision made by [Kubo](https://github.com/Kuborros), for [Ses Aeon](https://bsky.app/profile/sesaeon.bsky.social).

It includes their character _Lighting_ as a playable character in Freedom Planet 2!

### Features:

- Custom character art and sprites!
- Custom voices!
- Two custom music tracks!
- Few extra badges!
